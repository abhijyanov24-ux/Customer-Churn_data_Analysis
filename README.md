# Customer Churn Analysis

## 📌 Project Overview
This project focuses on analyzing customer churn data to understand why customers leave a company and identify key factors influencing churn behavior.

## 🎯 Objectives
- Analyze overall churn rate
- Identify factors affecting customer churn
- Understand churn based on contract type and monthly charges
- Discover patterns in customer behavior

## 🛠 Tools & Technologies Used
- Python (Pandas, NumPy)
- SQL
- Power BI

## 📊 Key Insights
- Customers with month-to-month contracts have higher churn rates
- Higher monthly charges are linked with increased churn
- Long-term customers are less likely to churn
- Certain customer segments show higher churn behavior

- Dataset (CSV file)
